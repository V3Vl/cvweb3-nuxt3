# Nuxt3+vite+pinia+antdV
# nodejs+express

# 安装

```shell
# yarn
yarn install
#pnpm
pnpm install
```

# 本地api
进入server文件夹
```shell
nodemon app.js

本地接口服务配置在`http://localhost:8081`

```shell
npm run dev
```

# 打包生产环境

将你的代码进行打包

```shell
npm run build
```

预览生产环境项目

```shell
npm run preview
```
the ready  template，local service is already configured.
